# Image Upscaling Service

Сервис для увеличения разрешения изображений с помощью модели EDSR (x2).  
Реализован на Flask + Celery + Redis.

## Возможности

- Асинхронная обработка изображений через Celery.
- Поддержка форматов: PNG, JPEG, BMP (на входе), выход всегда PNG.
- Три эндпоинта:
  - `POST /upscale` — загрузить изображение, получить ID задачи.
  - `GET /tasks/<task_id>` — узнать статус и ссылку на результат.
  - `GET /processed/<filename>` — скачать обработанный файл.
- Модель загружается один раз при старте воркера.

## Технологии

- Python 3.9+
- Flask
- Celery
- Redis (брокер и бэкенд результатов)
- OpenCV, NumPy
- Модель EDSR_x2.pb
